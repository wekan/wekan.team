Install


Login help

If you can not login, please email support@wekan.team from your login email address. Or alternatively, try the tips below.

a) Android: With `Play Store` install `Chrome` and `WeKan`, with Android `⚙ Settings / Default Settings` set `Chrome` as default browser, with `Chrome` login to [https://mail.google.com](https://mail.google.com), with `WeKan` click `Login with Google`. After that, `WeKan` starts fullscreen.

b) iOS: With `App Store` install `Chrome`, with `Chrome` login to [https://mail.google.com](https://mail.google.com), with `Chrome` at [https://boards.wekan.team](https://boards.wekan.team) click `Login with Google`.

c) Ubuntu Touch: With `OpenStore` install `WeKan`, with `Morph` browser login to [https://mail.google.com](https://mail.google.com), with `WeKan` click `Login with Google`.

d) Windows: With `MS Store` install `WeKan`, with `WeKan` click `Login with Google`.

e) Desktop browsers based on Safari/Firefox/Chromium: Login at [https://boards.wekan.team](https://boards.wekan.team). [Compatible Webbrowsers](https://github.com/wekan/wekan/wiki/Browser-compatibility-matrix).

f) Password: Write email address and password, and click `Sign In` at bottom.

g) Your own WeKan server URL: [Create your own app icon](https://github.com/wekan/wekan/wiki/PWA)
